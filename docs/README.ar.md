# django_toon — دليل التكامل مع Django

**django-toon** هي حزمة لإرسال البيانات بصيغة **Toon** والتي تعتبر `أسرع بكثير من JSON`.

توفر هذه المكتبة ميزتين أساسيتين لإطار **Django**، حيث يمكنك إنشاء Views `ديناميكية` تقوم بإرجاع `JSON` أو `Toon` حسب الطلب.

## التثبيت

```bash
pip install "django-toon"
```
